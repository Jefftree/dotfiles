# dotfiles

Configurations for Arch Linux cross compatible with OS X.

![Doge](http://i.imgur.com/5N3IEq2.jpg)

## Main Features
- vim
- zsh
- tmux

## Other
- pentadactyl
- i3
- urxvt
- Useful scripts in `/bin`

To run install script, run `install.sh`

Install script symlinks everything to home directory.
